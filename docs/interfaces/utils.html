remove

  • remove(handler: EventHandler<any>): void
    • Defined in event-delegation.d.ts:421

    A pure function that provides an alternative to calling the EventHandler.remove method. This can be convenient for function composition and other patterns.

    example
    declare const handlers: EventHandler<any>[]

handlers.forEach(EventDelegation.remove)
// is the same as
handlers.forEach((handler) => handler.remove())

    Parameters

    • handler: EventHandler<any>

    Returns void
